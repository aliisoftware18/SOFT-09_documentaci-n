# Guia de Funciones de Git y GitHub

Este es un proyecto individual desarrollado para la Universidad Cenfotec. El objetivo principal de esta herramienta es servir como un recurso educativo e interactivo para visualizar, comprender y repasar las funciones y comandos disponibles y mas utilizados en Git y GitHub.

---

## Descripcion del Proyecto

El proyecto nace de la necesidad de centralizar y explicar de forma clara el flujo de trabajo moderno con sistemas de control de versiones. Permite a los usuarios, especialmente a estudiantes de primer ingreso o de desarrollo de software, explorar desde los comandos basicos de terminal hasta los flujos colaborativos en la nube.

### Caracteristicas Principales
* **Glosario Interactivo:** Visualizacion detallada de comandos de Git (`init`, `clone`, `commit`, `push`, `pull`, etc.).
* **Flujos de Trabajo (Workflows):** Guias explicativas sobre como usar ramas (*branching*) y fusionar codigo (*merging/rebasing*).
* **Integracion con GitHub:** Explicacion de herramientas nativas como *Pull Requests*, *Issues*, *Projects* y *GitHub Actions*.
* **Interfaz Limpia:** Disenado de manera estructurada para un aprendizaje rapido y eficiente.

## Instalacion y Uso Local

Para explorar o ejecutar este proyecto de forma local, debe seguir los siguientes pasos:

1. **Clonar este repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/tu-repositorio.git
   ```

2. **Navegar al directorio del proyecto:**
   ```bash
   cd nombre-del-proyecto
   ```

3. **Visualizacion:**
   * Abra el archivo principal en su editor de codigo preferido o navegador web, segun la implementacion de la interfaz.
